Ściąga

  • Typy zmiennych:


    • int a = 1;
    Liczby całkowite

    float a = 1.1
    Liczby rzeczywiste

    double a = 1.11
    2x większa dokładność niż float

  • Predekrementacja:


    • --a; to a = a - 1;

  • Postdekrementacja:


    • a--; to a = a - 1;

  • Preinkrementacja:


    • ++a; to a = a + 1;

  • Postinkrementacja:


    • a++; to a = a + 1;

  • Operatory:


    • a -= 10; to a = a - 10;
    a += 10; to a = a + 10;
    a *= 10; to a = a * 10;
    a /= 10; to a = a / 10;
